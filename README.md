# CDocs

Lets you get the notification from Firebase server, and open up a document (i.e PDF file) link sent from server.

#### Helpers ::smiley::
- Firebase Cloud Messaging
- Twitter REST APIs
- RxJava/ RxAndroid
- Retrofit
- SQLite Database
- Preferences
- RecyclerView
- Stetho (For local data check)
- Glide


#### Structure
- MVP with android databinding
